# Contact Manager (Version 1)

A simple command-line contact manager written in Python.  
This project allows users to add, view, search, edit, and delete contacts, with data persistence using a text file.

---

## Features

- Add new contacts (name, phone, email)
- Prevent duplicate contacts
- Show all contacts (sorted alphabetically by name)
- Search contacts by name
- Edit existing contacts
- Delete contacts with confirmation
- Save contacts to a file
- Load contacts automatically on program start
